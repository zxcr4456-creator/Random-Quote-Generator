Краткое описание программы
Random Quote Generator - это десктопное приложение с графическим интерфейсом на Python для генерации и управления коллекцией цитат. Программа позволяет:

🎲 Генерировать случайные цитаты из базы данных

➕ Добавлять новые цитаты с проверкой корректности ввода

📜 Просматривать историю всех сгенерированных цитат

🔍 Фильтровать историю по автору и теме

💾 Автоматически сохранять все данные в JSON-файл

📊 Отслеживать статистику (количество цитат, авторов, тем)

Программа имеет интуитивно понятный интерфейс на русском языке и подходит для ежедневного использования.

Инструкция по запуску
Системные требования
Операционная система: Windows / macOS / Linux

Python: версия 3.7 или выше

Дополнительные библиотеки: не требуются (используются только стандартные модули Python)

Установка и запуск
1. Проверка установки Python
Откройте терминал (командную строку) и выполните:

bash
python --version
или

bash
python3 --version
Если Python не установлен, скачайте его с официального сайта

2. Скачивание программы
Вариант А: Клонирование через Git

bash
git clone https://github.com/username/random-quote-generator.git
cd random-quote-generator
Вариант Б: Прямая загрузка

Скачайте файл quote_generator.py из репозитория

Сохраните его в отдельную папку

3. Запуск программы
В терминале перейдите в папку с программой и выполните:

Для Windows:

bash
python quote_generator.py
Для macOS/Linux:

bash
python3 quote_generator.py
Точная команда для запуска:

bash
python quote_generator.py
или если Python 3 установлен как python3:

bash
python3 quote_generator.py
4. Первый запуск
При первом запуске программа автоматически создаст файл quotes_data.json в той же папке с предустановленными цитатами. Никаких дополнительных действий не требуется.

Возможные проблемы и решения
Проблема	Решение
ModuleNotFoundError: No module named 'tkinter'	На Linux: sudo apt-get install python3-tk
Ошибка с правами доступа	Запустите терминал от имени администратора
Не открывается окно	Проверьте, что файл quote_generator.py находится в текущей папке

text

- Фильтры обновились с учётом новых авторов и тем
Пример 8: Горячая клавиша F5
